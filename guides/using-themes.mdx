Themes control how agents are named in Proletariat. Choose from built-in themes or create your own.

## Why Themes?

Agent names make your workspace more:
- **Memorable**: "bezos" is easier than "agent-7"
- **Fun**: "Finally, they work for us" 😄
- **Organized**: Themes help distinguish agent types
- **Scalable**: Built-in themes have 50+ names each

## Managing Themes

### List Themes

View all available themes:

```bash
prlt agent themes list
```

Output:

```
Agent Themes

  Billionaires [built-in]
    ID: billionaires
    The ultra-wealthy work for you
    Names: 53 available, 0 in use

  Toyota Garage [built-in]
    ID: toyotas
    Reliable workhorses for your project
    Names: 72 available, 0 in use

  Company Portfolio [built-in]
    ID: companies
    Your corporate portfolio
    Names: 96 available, 0 in use
```

### Set Active Theme

Change the active theme:

```bash
prlt agent themes set toyotas
```

New agents will use names from the selected theme.

<Note>
Changing theme doesn't rename existing agents. Only new agents use the new theme.
</Note>

### View Theme Names

See all names in a theme:

```bash
prlt agent themes list --json | jq -r '.[] | select(.id == "billionaires") | .availableNames'
```

## Custom Themes

Create your own themes:

<Steps>

<Step title="Create the theme">

```bash
prlt agent themes create mytheme --name "My Theme" --description "My custom agent names"
```

</Step>

<Step title="Add names">

```bash
prlt agent themes add-names mytheme
```

You'll be prompted to enter names one by one. Press Ctrl+D when done.

Or add from a file:

```bash
cat names.txt | prlt agent themes add-names mytheme
```

Example `names.txt`:

```text
alpha
bravo
charlie
delta
echo
foxtrot
```

</Step>

<Step title="Set as active">

```bash
prlt agent themes set mytheme
```

</Step>

</Steps>

## Agent Naming Rules

### Staff Agents

**Use theme names directly:**

```bash
prlt agent staff add bezos
```

Agent is named `bezos`.

### Ephemeral Agents

**Prefix with adjective:**

When creating ephemeral agents:

```bash
prlt work start TKT-042 --ephemeral
```

Agent is auto-named like:
- `bold-bezos-1`
- `keen-gates-2`
- `wise-musk-3`

**Format:** `<adjective>-<theme-name>-<number>`

<Note>
The number suffix (e.g., `-1`) will be removed in a future version. Names will be just `bold-bezos`, `keen-gates`, etc.
</Note>

### Name Validation

Agent names must:
- Start with letter or number
- Contain only: `a-z`, `A-Z`, `0-9`, `-`, `_`
- Be unique (case-insensitive)

Invalid names:
- `my agent` (spaces not allowed)
- `agent@123` (@ not allowed)
- `_private` (can't start with underscore)

## Theme Selection During Setup

When running `prlt new`, you'll be prompted to choose a theme:

```
? Select agent naming theme:
  › Billionaires - The ultra-wealthy work for you
    Toyota Garage - Reliable workhorses
    Company Portfolio - Your corporate portfolio
    Custom... - Create your own
```

This sets the workspace default theme.

## Changing Themes

### Switch Themes

```bash
prlt agent themes set companies
```

**What happens:**
- ✅ New agents use new theme
- ✅ Old agents keep their names
- ✅ Theme setting is saved to workspace

### Rename Existing Agents

Themes don't auto-rename agents. To rename:

```bash
# Remove old agent
prlt agent staff remove bezos

# Add new agent with desired name
prlt agent staff add stripe
```

<Warning>
Removing an agent deletes its workspace. Back up any uncommitted work first.
</Warning>

## Theme Exhaustion

**What if all names are taken?**

When all theme names are in use:
- Ephemeral agents append numbers: `bold-bezos-4`
- Staff agents require manual names
- Consider adding more names to theme
- Or create additional custom theme

**Check availability:**

```bash
prlt agent themes list
```

Shows available vs. in-use names per theme.

## Best Practices

<AccordionGroup>

<Accordion title="Choose theme at setup">
  Pick your theme during `prlt new`. Changing later doesn't rename existing agents.
</Accordion>

<Accordion title="Keep it fun">
  Themes make your workspace more enjoyable. Don't stress about being professional—embrace the fun.
</Accordion>

<Accordion title="Use descriptive themes for teams">
  If sharing workspace with a team, choose themes that resonate with everyone.
</Accordion>

<Accordion title="Create project-specific themes">
  For domain-specific projects, create custom themes that match your context (e.g., `planets` for space project).
</Accordion>

<Accordion title="Add plenty of names">
  If creating custom themes, add at least 20-30 names to avoid exhaustion.
</Accordion>

